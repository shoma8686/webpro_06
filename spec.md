# Webプログラミング 最終課題 仕様メモ

## 1. 書籍管理アプリ (book_list)
* **変数名:** `book_list`
* **URLプレフィックス:** `/books`
* **データ構造:**
    * `id` (数値): ID
    * `title` (文字列): 書名
    * `author` (文字列): 著者
    * `price` (数値): 価格
    * `publisher` (文字列): 出版社

## 2. 商品管理アプリ (product_list)
* **変数名:** `product_list`
* **URLプレフィックス:** `/products`
* **データ構造:**
    * `id` (数値): ID
    * `code` (文字列): 商品コード
    * `name` (文字列): 商品名
    * `category` (文字列): カテゴリ
    * `price` (数値): 価格

## 3. ユーザー管理アプリ (user_list)
* **変数名:** `user_list`
* **URLプレフィックス:** `/users`
* **データ構造:**
    * `id` (数値): ID
    * `userid` (文字列): ユーザーID
    * `name` (文字列): 名前
    * `age` (数値): 年齢
    * `email` (文字列): メールアドレス
